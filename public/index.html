自动刷新时间窗口

订阅按每日固定窗口计算刷新点。例如 08:00 开始、间隔 120 分钟，会在 08:00、10:00、12:00 依次刷新。

Embedding 检索

配置 OpenAI 兼容的 /embeddings 接口。未开启或配置不完整时，会继续使用简单关键词搜索。

AI 资料整理提示词

按资料来源配置整理规则。资料详情页点击生成整理后，会使用对应类型的提示词处理原文。

页面来源认证

抓取会优先调用 Jira、Confluence、GitHub 或 Microsoft Graph API，API 不可用时再回退到 HTML/WebDriver。Token、Cookie 或 Basic 认证仅保存在本机。
